Реализуйте билдер для класса автомобиля.
Поля автомобиля:
    - *Марка: String
    - *Год выпуска: String
    - Владелец: String (по умолчанию пустая строка)
    - Цвет: String/enum (по умолчанию черный)
    - Кол-во колес: Integer (по умолчанию 4)
    - Пробег: Integer (по умолчанию 0)
    - Кол-во дтп с участием авто: Integer (по умолчанию 0)

Марка и год выпуска являются обязательными полями и не имеют значения
по умолчанию.
(Можно вытащить их в конструктор)
